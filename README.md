# cctv-footage-datetime-ocr

## For initial setup
Create a venv then activate
```
python venv -m .venv
```
Activate the created venv and install the required packages
```
source .venv/bin/activate
pip install -r requirements.txt
```

## For running the jupyter notebook
Activate the venv (if not yet activated)
```
source .venv/bin/activate
```
Run the jupyter notebook
```
jupyter notebook
```
